Hướng dẫn chạy source code:
- Cài đặt các thư viện cần thiết bằng lệnh:
pip install -r requirements.txt
- Chạy file main.py để chạy các thuật toán với các file input
trong folder inputs đã được cài đặt sẵn, kết quả chạy các thuật
toán sẽ được lưu vào các file output tương ứng trong folder outputs.
- Chạy file GUI.py để mở giao diện và thực hiện các thao tác 
với giao diện
